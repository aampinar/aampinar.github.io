Revisions


I spoke with my colleague Kris Roldan and personal friend, Tyson Walton, who both provided me very constructive feedback to my site. Both provided visual and mechanical update recommendations to give my site better viewability and optimization. The following section contains the changes I made to my site.

Correction #1

The first correction I made was in relation to my tool bar. It wasn't sticking to the top of the screen when a user would scroll down. I remedied this by changing the position of the nav style to fixed, shifting the top gap to 0 and adding javascript to incorporate the "sticking effect" when a page reaches it's minimum scrolling requirement.

Correction #2

The second correction I made was in relation to wayfinding. Previously, when navigating the screen the home button on the navigation bar would remain blue in an "active" status. Now when navigating each screen, the screen visted is highlighted in the nav bar.

Correction #3

The third correction entailed the ostrich logo and the screen loading. Previously when clicking on the link, it would send the user to the linked site and leave the page. With the correction, it now opens a separate tab when loading the link and keeping the user on the site.
